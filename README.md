## 🚀 Quick Start

### **Prerequisites**
```bash
Python 3.9+
pip install -r requirements.txt
```

### **Installation**
```bash
# Clone the repository
git clone https://github.com/RahuulPande/talent-edge-crm-toolkit.git
cd talent-edge-crm-toolkit

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```

### **Mobile Access**
1. Open the app in your browser
2. Navigate to the Introduction page
3. Scan the QR code with your mobile device
4. Access all features on mobile

## 🔧 Configuration

### **Environment Setup**
```bash
# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### **Streamlit Configuration**
```toml
# .streamlit/config.toml
[server]
port = 8501
address = "0.0.0.0"

[browser]
gatherUsageStats = false

[theme]
primaryColor = "#1f77b4"
backgroundColor = "#ffffff"
secondaryBackgroundColor = "#f0f2f6"
textColor = "#262730"
```
